아이폰 실전 연습

정답 듣기 → 내 목소리 녹음 → 재생 비교 방식으로 연습합니다.

의전 영어 v1.1.0

아이폰 사용 방법

홈 화면 설치: Safari 하단 공유 버튼 → 홈 화면에 추가

발음 듣기: 🔊 버튼을 누른 뒤 속도를 선택하세요.

내 목소리 녹음: 🎙️ 버튼 → 녹음 시작 → 문장 말하기 → 녹음 중지 → 재생

통역 연습: 한국어를 보고 영어로 말한 뒤 정답을 확인합니다.

쉐도잉: 영어 음성을 듣고 설정한 대기 시간 동안 따라 말합니다.

마이크 사용 시 Safari가 권한을 요청하면 허용을 눌러주세요.

학습 원칙: 전문 통역 표현보다 현장에서 바로 사용할 수 있는 짧고 쉬운 영어를 우선합니다.

긴 문장이 부담되면 카드의 외우기 쉬운 표현부터 연습하세요.

업데이트: 새 버전이 감지되면 화면에 업데이트 버튼이 나타납니다.

현재 버전: v1.1.0
